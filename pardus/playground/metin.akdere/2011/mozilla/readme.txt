Firefox 12
==========

* Build nss, nspr, and libvpx packages before building Firefox 12.

* You can find the PiSi binaries for x86_64 machines at http://cekirdek.pardus.org.tr/~metin/dist/bin. Again, do not forget to install the latest nss, nspr, libvpx packages first from the same URL before installing Firefox 12.
